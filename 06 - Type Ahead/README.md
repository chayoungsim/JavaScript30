# 입력창에 타이핑한 값에 따라 도시/주 이름을 실시간으로 필터링하여 목록으로 보여주는 검색 기능의 전체 흐름을 구현한 예제

### 전체 동작 흐름 요약
1. 페이지 로드 → 도시 데이터 fetch
2. 사용자가 입력창에 타이핑
3. displayMatches() 실행
4. 검색어와 일치하는 도시 필터링
5. 일치 텍스트 하이라이트
6. li 목록으로 화면에 출력

### 사용자 입력값(word)을 정규식으로 만들려면 생성자 방식이 필수입니다.
- new RegExp('abc', 'gi');
- escapeRegExp(word) - 사용자 입력 보호
- RegExp(..., 'gi') - 대소문자 무시 검색
- filter() - 조건에 맞는 데이터 추출
- match() - 포함 여부 검사

### replace()  - 문자열을 다른 문자열로 교체
- place.city.replace(regex, `<span>$&</span>`);
- 실제 매칭된 문자열 그대로 사용
- 가장 안전하고 표준적인 방법
